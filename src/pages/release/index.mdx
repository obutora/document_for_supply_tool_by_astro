# Release
ツールの一覧です。  
便利なツールがたくさんありますが、どこにあるか、  
だれに聞けばいいか分からない迷子の方が増えている...  
かもしれませんので、こちらのページにまとめておきます。  
保存場所は頻繁に変わるため、分からない場合は担当者に連絡してみてください。

| ツール名          | 担当者         | 説明                                                               |                                                                                
| ---------------- | ------------- | ------------------------------------------------------------------- | 
| OTC可視化ツール   | 芳賀<br/>(たかお店)|OTCやキャンペーンの売上を美しく可視化し、<br/>モチベーションを高めるツールです。                                                                                                                                           |
| Hakasu<br/>(DS管理ツール)   | 芳賀<br/>(たかお店)|出庫量をもとにDSの引き取り先を検討し、<br/>自動で依頼書を作成するツールです。
| 発注点ツール  | 堀<br/>（大洋堂）| 過去の出庫量に基づいて発注点を自動で設定し、<br />発注時間を5~10分に短縮するツールです。
