# 発注点設定ファイルを作成する

<img src="../../../../document_for_supply_tool_by_astro/1_3.png" />{" "}

1. ツールシートで **⓪ 初期化**を押す
2. **出庫更新**を押す
   - 準備した入庫・出庫実績一覧を選択
   - 複数ある場合は、古いものから選択する。
   - 新しいものを先に選んだ場合、⓪ からやり直す
3. **発注点取込**を押す
   - 準備した年間発注点設定を選択
4. **発注点設定 → 出力**を押す
   - 本ツールと同じフォルダに **【出力】発注点.csv/** が作成されます。
   - ※ 初回分はフラグが反映されていないので、年間発注点 取込には**使用しない**
5. フラグシートで薬品名を指定してフラグを手作業で設定
   - 詳細は[フラグの操作説明](/document_for_supply_tool_by_astro/supplyTools/2_basic/2_1)で後述
6. **発注点設定 → 出力**を押す
   - 本ツールと同じフォルダに **【出力】発注点.csv が作成される**
7. CHOIS に発注点を取り込む
