# 発注点設定ファイルを作成する

<img src="../../../../1_3.png" />{" "}

1. ツールシートで **⓪ 初期化**を押す
2. **出庫更新**を押す
   - 準備した入庫・出庫実績一覧を選択
   - 複数ある場合は、古いものから選択する。
   - 新しいものを先に選んだ場合、⓪ からやり直す
3. **発注点取込**を押す
   - 準備した年間発注点設定を選択
4. **発注点設定 → 出力**を押す
   - 本ツールと同じフォルダに **【出力】発注点.csv/** が作成されます。
   - 初回分は破棄するのがお勧め
5. フラグシートで薬品名を指定してフラグを手作業で設定
   - 詳細はフラグの操作説明で後述
6. ツールシートでメンテナンスを押す
   - フラグの注意事項が出るので、必要に応じてフラグ修正
   - CHOIS で発注 JAN 指定して年間発注点設定出力からやり直す
7. **発注点設定 → 出力**を押す
   - 本ツールと同じフォルダに **【出力】発注点.csv が作成される**
8. CHOIS に発注点を取り込む
   - CHOIS
     - 発注
       - 年間発注点設定
         - 取込
           - **出力】発注点.csv** を選択
             - 登録
