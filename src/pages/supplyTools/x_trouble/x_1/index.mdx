# マクロが動かない
現在、WindowsUpdate後にエクセル等を開くと  
マクロに関するエラーが発生する事象が発生しているとの報告があります。
 
> 例）
> マクロが入っているExcelを開いたところ、「このブックにある、VBA プロジェクト, ActiveX コントロール,
> およびその他のプログラミング関連の機能は失われています。」のエラーが出る
 
以下の暫定対応実施し、改善するか確認をお願いいたします。
 
### 手順
<img src="../../../../document_for_supply_tool_by_astro/office_dialog.png" />
1. コントロールパネル
2. プログラム
3. プログラムと機能
4. Microsoft office standard 2016を右クリック
5. 変更
6. 機能の追加/削除　←←！注意！「削除」は選択しない
7. 不具合が起きているエクセルやワードそれぞれで、「マイコンピュータから全て実行」をクリック
8. 続行をクリック
