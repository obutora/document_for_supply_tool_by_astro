# ダウンロード
このページでは、発注点ツールをダウンロードすることが出来ます。  
ダウンロードした後、**右クリック** → **全て展開**してお使いください。

## 前のVerのデータを引き継ぐ方法
最新版のツールシートにある**旧データ取り込み**ボタンを押し、  
現在使用している発注点ツールを選択するだけでフラグなど主要なデータのほとんどを**引き継ぐ**ことが出来ます。
